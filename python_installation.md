# README – Installation von Python 3.12.7 (Einsteigerfreundlich)

Diese Anleitung erklärt Schritt für Schritt, wie **Python 3.12.7** auf **Windows** und **macOS** installiert wird.  
Sie kommt **ohne Terminal oder Kommandozeile** aus.

---

## 1. Python 3.12.7 herunterladen

Nutze die offizielle Download-Seite von Python:

**https://www.python.org/downloads/release/python-3127/**

Scrolle auf der Seite nach unten bis zum Abschnitt **Files**.

Dort erscheint eine lange Liste von Dateien — aber du brauchst **nur eine einzige**, abhängig von deinem Betriebssystem.

---

## 2. Installation unter Windows

### Welche Datei herunterladen?
Suche nach der Datei:

**Windows installer (64-bit)**  
Beispieldatei: `python-3.12.7-amd64.exe`

Klicke darauf, um die Datei herunterzuladen.

### Installation
1. Öffne die heruntergeladene Datei (`.exe`).
2. Ein Installationsfenster erscheint.
3. ❗❗**Wichtig:** Setze das Häkchen bei  
   **Add Python to PATH**. .❗❗
4. Klicke auf **Install Now**.
5. Warte, bis die Installation abgeschlossen ist.
6. Schliesse das Fenster.

### Überprüfen, ob Python installiert wurde
1. Öffne das **Startmenü**.
2. Gib **Python** ein.
3. Du solltest nun **Python 3.12** und **IDLE (Python GUI)** sehen.

---

## 3. Installation unter macOS

### Welche Datei herunterladen?
Suche nach der Datei:

**macOS installer**  
Beispieldatei: `python-3.12.7-macos11.pkg`

Klicke darauf, um sie herunterzuladen.

### Installation
1. Öffne die `.pkg`-Datei.
2. Ein macOS-Installationsassistent startet.
3. Klicke auf **Fortfahren**, **Fortfahren**, dann **Installieren**.
4. Gib dein Mac-Passwort ein (falls gefragt).
5. Warte, bis die Installation abgeschlossen ist.

### Überprüfen, ob Python installiert wurde
1. Öffne den **Finder**.
2. Gehe zu **Programme**.
3. Du findest einen Ordner namens **Python 3.12**.
4. Öffne ihn und starte **IDLE** (ein einfacher Python-Editor).

---

## 4. Zusammenfassung

| Betriebssystem | Datei zum Herunterladen | Beschreibung |
|----------------|--------------------------|--------------|
| **Windows**    | `python-3.12.7-amd64.exe` | Standard-Installer, einfache Installation |
| **macOS**      | `python-3.12.7-macos11.pkg` | Standard-Installer für Mac |

**Hinweis:**  
Lade **nicht** die Dateien mit der Bezeichnung *source tarball* herunter.  
Diese sind nur für Entwickler gedacht, die Python aus dem Quellcode kompilieren möchten.

---

## 5. Nächste Schritte

Nach der Installation kannst du:

- **IDLE** öffnen und deinen ersten Python-Code schreiben.
- Python in einem Editor wie VS Code oder PyCharm nutzen.
- Mit dem Projekt in diesem Repository starten.
